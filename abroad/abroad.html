Gallery Example

I am working on this on Google Cloud using a framework

I was hoping to use the Google Photos API to use images in my collection, instead of using links. It is still a work in progress, so this is a different approach for now.

Picker API

The Google Photos API is now divided into two, Picker and Library.

I was able to set up an auth process to allow me to access my library and utilize the Picker flow.

Library API

I was struggling with the Library API, proper authentication, and accessing photos. This current API is meant to limit access to these photos from the API to photos that the app itself has created. I was able to properly post new albums, without any media. But was running into issues accessing albums, even though it was app created, and adding media.

I plan to keep working on this, but for now have set up a gallery that uses manually provided links instead of using a GET request from an album like I had hoped.
